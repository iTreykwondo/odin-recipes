# Odin Recipes

This is the first project in [The Odin Project](https://www.theodinproject.com/paths/foundations/courses/foundations/lessons/recipes). The goal of this project was to get comfortable using HTML lists, different HTML elements, as well as linking between multiple HTML pages. Using CSS to add basic styles and layout.

Live preview can be found [here](https://itreykwondo.github.io/odin-recipes/)
